 Memory Card Game

A simple and interactive Memory Card Matching Game built using HTML, CSS, and JavaScript.
The goal is to match all pairs of cards. Each reload or reset randomizes the card order for a fresh experience every time!

 How to Play

Match all pairs to win the game!
